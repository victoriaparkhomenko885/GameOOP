Чтоб запустить файл, вам необходимо
иметь установленный Python-интерпритатор на своем устройстве.

1 Вариант:
- Откройте командныю строку (cmd)
- Впишите
py <path>
<path> - это путь к файлу main.py на вашем устройстве
Возможны также такие варианты:
python <path>
python3 <path>

2 Вариант:
Запустите файл main.py из установленного IDE на
вашем устройстве, например, IDLE или PyCharm

Затем наслаждайтесь игрой, правила описанны в самой игре.
